# News Intelligence Bundle

Product class: `Intelligence Bundle`

Independent Intelligence Bundle for Manager-defined News Topics, bounded Core-owned collection,
source-backed event/claim/correction modeling, cited briefings, Penny context and reviewed graph
connections. Topic authority remains the Core Knowledge collection plane; this package owns only
derived News domain records.

The runtime has no network egress. It uses the typed Bundle broker for tenant-forced PostgreSQL
records and its own signed collection profile.
