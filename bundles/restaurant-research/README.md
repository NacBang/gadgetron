# Restaurant Research Bundle

> Product class: `Intelligence Bundle`

Independent signed `restaurant.*` runtime for cited branch, menu, review,
recommendation and visit-outcome records. External URLs are ingested only by
the Core Source-to-Vault path; this runtime has no network egress.
Manifest v3 declares this package as `Intelligence`; booking, payment and
itinerary mutation remain Operational responsibilities outside this package.

Build with `scripts/build-package.sh`. Use `scripts/stage-dev-package.sh` only
for the 18085 development package catalog.
