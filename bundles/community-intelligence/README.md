# Community Intelligence

> Product class: `Intelligence Bundle`

Independent Intelligence Bundle for cited discussion, answer and solution-pattern knowledge.
The runtime owns `community.*` domain records and has no network access. Core owns official
provider queries, credentials, rate limits, Source snapshots, Review, Graph projection and Penny
execution.

Stack Exchange collection is usable through the official v2.3 API. Reddit remains fail-closed
until Core has both approved API access and a Source store that can purge deleted content from
historical storage.
