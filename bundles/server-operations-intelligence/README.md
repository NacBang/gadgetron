# Server Operations Intelligence Bundle

> Product class: `Intelligence Bundle`

Independent signed ontology, collection, and AI-role package for vendor guidance,
community cases, runbooks, contradictions, and outcome-backed server lessons. It
does not register, monitor, or modify servers and holds no SSH, secret, or direct
network permission. Its database grant is limited to reading operational subject
revisions and attaching provider-owned enrichment rows. Core owns source fetching,
event state, and Knowledge lifecycle.

Build with `scripts/build-package.sh`. Use `scripts/stage-dev-package.sh` only for
the 18085 development package catalog.
