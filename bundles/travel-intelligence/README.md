# Travel Intelligence Bundle

> Product class: `Intelligence Bundle`

Independent signed ontology, collection, and AI-role package for destinations,
options, advisories, contradictions, and outcome-backed travel knowledge. It
does not own or mutate Trips, itineraries, budgets, or reservations. Core owns
source fetching and Knowledge lifecycle.

Build with `scripts/build-package.sh`. Use `scripts/stage-dev-package.sh` only
for the 18085 development package catalog.
